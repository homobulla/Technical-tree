    # 对象的操作与描述

    在JS中，简单基本类型并不是对象，这里的null有时被当做对象类型，但本质上null是基本类型。

    JS中有一些对象子类型，通常被称为内置对象。

    • String
    • Number
    • Boolean

    # 对象的拷贝

    • 浅拷贝： 以赋值的形式拷贝引用对象，扔指向同一个地址，修改原对象会造成同步修改
      • Object.assign
      • 展开运算符
    • 深拷贝
      • JSON.parse(JSON.stringify(obj)):性能最快但又局限性，在具有新欢引用对象时报错，当值为函数、undefined\Symbol时无法拷贝
      • 递归进行逐一赋值

    # 属性描述符

    Object.getOwnProertyDescriptor作为对象的属性描述符，会返回四个值：value(值) writable(可写) enumerable(可枚举) configurable(可配置)

    let obj = {name:'homo'};
Object.getOwnPropertyDescriptor(obj,'name');
//{value: "homo", writable: true, enumerable: true, configurable: true}

    writable enumerable configurable默认值为true，可以通过defineProperty进行修改。

    Object.defineProperty(obj,'name',{
    value:'bulla',
    writable:false})
Object.getOwnPropertyDescriptor(obj,'name');
//{value: "bulla", writable: false, enumerable: true, configurable: true}

    # 不变性

    结合 writable:false 和 configurable:false 就可以创建一个真正的常量属性（不可修改、重定义或者删除）：

    let obj = {}
Object.defineProperty(obj,"NAME",{
    value:'删不掉，改不动，气不气',
    writable:false,
    configurable:false
})

    # setter getter

    我们可以通过这两个方法来修改对象的默认行为：

    let obj = {
    get a(){
        return this._a
    },
    set a(val){
    	this._a = val * 3
	}
}
obj.a = 10
obj.a // 30
