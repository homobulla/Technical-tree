# Notification

Notification 是 HTML5 新增的 API，用于向用户配置和显示桌面通知。

# 特性

  • 该通知是脱离浏览器的，即使用户没有停留在当前标签页，甚至最小化了浏览器，也会在主屏幕的右上角显示通知，然后在一段时间后消失。
  • 我们可以监听通知的显示，点击，关闭等事件,比如点击通知打开一个页面。

# 使用

一个很简单的 API：

var options = {
    dir: 'auto', // 文字方向
    body: '通知：OBKoro1评论了你的朋友圈', // 通知主体
    requireInteraction: true, // 不自动关闭通知
    // 通知图标
    icon:
        'https://upload-images.jianshu.io/upload_images/5245297-818e624b75271127.jpg?imageMogr2/auto-orient/strip%7CimageView2/2/w/1240'
}
notifyMe('这是通知的标题', options)

# 应用场景

  • 即时通讯软件(邮件、聊天室)

  • 新闻网站重大新闻通知
  • 网站的重大更新，重大新闻等。
